Overview

        This Java application lets users upload a CSV file, pick a numeric column, and compare the execution time of five sorting algorithms.
        The results are shown through a simple GUI and a bar chart.

Features

        Upload a .csv file
        Extract and display numeric columns

Run 5 sorting algorithms:

      Insertion Sort
      Shell Sort
      Merge Sort
      Quick Sort
      Heap Sort

Measure execution time in milliseconds

      Show best algorithm
      Display results 


 How to Use

    Run Main.java.
    Click Upload CSV File.
    Select a numeric column from the dropdown.
    Click Run Performance Evaluation.

View:

    Best algorithm
    Execution times for all algorithms
